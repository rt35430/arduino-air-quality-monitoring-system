Overview

This project uses an Arduino Uno R3, MQ135 air quality sensor, and DHT11 temperature sensor to monitor environmental conditions in real time.

The project was created for a science fair project focused on environmental sustainability and air pollution awareness.

Features
Real-time air quality monitoring
Temperature and humidity readings
OLED display output
Air pollution awareness
Instagram awareness campaign
Components Used
Arduino Uno R3
MQ135 Sensor
DHT11 Sensor
OLED Display
Breadboard
Jumper Wires
Testing

The system was tested using:

Ignited tissue
Lighter smoke
Isopropyl alcohol
Purpose

The goal of this project is to help spread awareness about air pollution and environmental health using low-cost technology.
